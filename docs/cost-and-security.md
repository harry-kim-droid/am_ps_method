# 비용·보안 설정

모델 가격은 변경될 수 있으므로 OpenAI 공식 가격표를 확인하고 저장소별 월 예산을 정합니다. PR 번호, 실행 횟수, 모델, 유효한 지적 수를 주간 단위로 기록하세요.

## 비용 절감 순서

1. 초안 PR과 문서 전용 변경을 제외합니다.
2. 큰 PR을 작은 변경으로 나눕니다.
3. 필요하면 자동 실행을 라벨 또는 수동 실행으로 바꿉니다.
4. 저장소 변수 `CODEX_MODEL`에는 실제 계정에서 사용 가능한 모델만 설정합니다.

## 보안 기준

- `OPENAI_API_KEY`는 GitHub Actions Secret에만 저장합니다.
- Codex 단계는 `contents: read`, `sandbox: read-only`를 유지합니다.
- 댓글 작성 권한은 별도 job에만 부여합니다.
- 고객 데이터, 프로덕션 로그, 자격 증명을 프롬프트에 넣지 않습니다.
- fork PR에 Secret이 전달되지 않는 GitHub 기본 보호를 해제하지 않습니다.
- 첫 실행 후 로그와 artifact에 민감정보가 없는지 확인합니다.
