# Codex Review Desk for Small Teams

## Faster PR feedback. Cleaner weekly engineering priorities.

Small teams move fast, but code review often depends on one overloaded senior engineer. Codex Review Desk gives your team an AI-assisted review layer for pull requests and a weekly technical debt report that turns scattered issues into a ranked action list.

## What You Get

- Pull request summaries for selected PRs
- Risk-focused review findings
- Missing test and edge-case suggestions
- Weekly technical debt report
- Ranked cleanup tasks for the next sprint
- Human-verified output during the pilot

## Best Fit

This is designed for teams that:

- Use GitHub pull requests
- Have 1-10 engineers
- Want more consistent review coverage
- Have technical debt but no dedicated platform team
- Want practical findings, not long generic AI reports

## Pilot

Duration:

- 2 weeks

Includes:

- 1 repository
- Up to 20 pull requests
- 1 weekly technical debt report
- Setup and review calibration

Pilot goal:

- Prove that the service finds useful issues and saves review time before a monthly subscription.

## Monthly Plans

Starter:

- 1-2 repositories
- Up to 50 PRs/month
- Weekly report
- KRW 300,000-700,000/month

Team:

- 3-5 repositories
- Up to 150 PRs/month
- Weekly report and monthly review call
- KRW 900,000-2,000,000/month

## Sample Weekly Output

- Top risk this week
- Top 5 technical debt items
- Suggested cleanup PRs
- Repeated review patterns
- Test coverage gaps
- Recommended next sprint actions

## Clear Boundary

This is not a replacement for your engineers. It is a review assistant and technical debt radar that helps your team spend less time finding the obvious issues and more time making product decisions.
