# 비용·보안 설정

## 비용

모델 가격은 변경될 수 있으므로 OpenAI 공식 가격 페이지를 확인합니다. 저장소별 월 예산을 정하고 PR 번호, 실행 횟수, 사용 모델, 유효한 지적 수를 기록하세요.

비용을 줄이는 순서:

1. 초안 PR과 문서 전용 변경을 제외합니다.
2. 큰 PR을 작은 변경으로 나눕니다.
3. 모든 PR 자동 실행 대신 라벨 또는 수동 실행으로 전환합니다.
4. `CODEX_MODEL` 저장소 변수로 검증된 저비용 모델을 선택합니다.

## 보안

- `OPENAI_API_KEY`는 GitHub Actions Secret에만 저장합니다.
- Codex 작업은 `contents: read`, `sandbox: read-only`로 실행합니다.
- 댓글 게시 권한은 별도 job에만 부여합니다.
- 고객 데이터, 자격 증명, 프로덕션 로그를 프롬프트에 넣지 않습니다.
- fork PR에 Secret이 전달되지 않는 GitHub의 기본 보호 동작을 약화하지 않습니다.
- 첫 실행 후 Actions 로그와 artifact에 민감정보가 없는지 확인합니다.
